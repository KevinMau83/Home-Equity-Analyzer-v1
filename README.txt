Homebuyer Equity Analyzer Web App

Files:
- index.html

How to use:
1. Upload the ZIP contents to Netlify, Cloudflare Pages, GitHub Pages, or any simple web host.
2. Or open index.html directly in a browser on a phone or computer.
3. Everything runs in the browser. No build step is needed.

What it does:
- Checks total repairs
- Checks total cost basis
- Calculates immediate equity
- Suggests max offer
- Checks renovation loan fit using 80% of ARV
- Gives a simple Healthy / Thin / Risky result

Note:
This tool is for planning and education. It does not replace lender, title, inspection, appraisal, contractor, tax, or legal advice.
